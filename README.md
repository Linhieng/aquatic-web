# 蓝色粮仓水产品冷藏运输

下面命令可以在 `bash` 命令行中操作，
也可以在 vscode 中操作。
如果是在 vscode 中操作，需要先进入 `bash`。

在 vscode 的终端直接输入
```bash
bash
```
就可以进入 `bash` 命令行了

1. 首先克隆仓库

```bash
git clone https://gitee.com/linhieng/aquatic-products-web.git
```

2. 然后进入仓库目录

```bash
cd aquatic-products-web
```

3. 安装依赖

```bash
npm i
```

4. 运行

```bash
npm run serve
```

或者直接在 vscode 的 NPM 脚本中点击 serve 运行
